# V2 Benchmark: V1 vs V2 性能对比

所有测试在单块 NVIDIA RTX 4090 上完成，bfloat16，`q_block_size = score_block_size`。V1 与 V2 使用等价参数（`threshold=1.0`，`qblock_topk = q_block_size * K`），输出完全一致。

## 打分（纯打分，不含注意力输出）

V1 在单个 Triton 内核中融合打分 + Bitonic Sort；V2 将 block scoring（D >= 80 时使用 CUDA CuTe MMA，否则 Triton）与 top-k selection（CUDA CUB 基数排序）分离。K=32 时差距最大（4.9x），因为 V1 的在线 Bitonic Sort 开销随 K 增长，而 V2 的 block scoring 与 K 无关。

## 端到端流水线（打分 + 合并 + 注意力）

V2 总耗时包含 `flash_block_score` + `flash_topk_select` + `flash_qblock_merge` + `flash_sparse_attn`。合并步骤（`flash_qblock_merge`）在所有场景下额外开销 < 0.2 ms。
